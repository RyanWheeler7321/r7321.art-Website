I spent several weeks experimenting with game ideas and trying to decide what the next main project should be.

I still had not fully committed to a single one, but a few directions were already strong enough to show off.

## bithunters

{% image "/images/updates/new-game-projects/bithunters-01.webp", "bithunters concept screenshot" %}

*bithunters* is a 2D platformer co-op action adventure game. Up to four players control personified futuristic computer programs and fight through virtual opponents together.

## Three Potions

{% image "/images/updates/new-game-projects/three-potions-01.webp", "Three Potions concept screenshot" %}

*Three Potions* is a 2D mission-based hack-and-slash action game with limited resources and difficult boss fights.

{% cta "https://store.steampowered.com/app/2324790/Three_Potions/", "Three Potions on Steam", "Wishlist page" %}

## Castle Mournfall

{% image "/images/updates/new-game-projects/castle-mournfall-01.webp", "Castle Mournfall concept screenshot" %}

*Castle Mournfall* is a 3D adventure game built around real-time combat, open-world exploration, and a larger story frame than the other experiments in the batch.

{% callout "note", "Working method" %}
The current strategy was to sacrifice single-project focus in order to maximize progress across multiple promising directions. Once one of them gets close, it can be sprinted to the finish line. That still looks like the most accurate description of how this body of work actually moves.
{% endcallout %}
